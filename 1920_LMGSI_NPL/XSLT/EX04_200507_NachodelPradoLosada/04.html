x64

DOCUMENTACIÓN:

PASOS A SEGUIR

P01

Paso 1

sudo apt update

Actualización del repositorio

P02

Paso 2

sudo apt install apache2

Instalación del servidor HTTP Apache2

P03

Paso 3

sudo service apache2 status

Comprobación del estado del servicio

P04

Paso 4

sudo adduser --home /var/www/html --group www-data operadorweb

Creación del usuario operadorweb

P05

Paso 5

sudo chmod -R 2775 /var/www/html

Cambiar los permisos de forma recursiva al directorio html, de forma que el usuario tenga todos los permisos, el grupo tenga todos los permisos y el resto lectura y ejecución

P06

Paso 6

sudo chown -R operadorweb:www-data /var/www/html

Cambiar el propietario de la carpeta html
